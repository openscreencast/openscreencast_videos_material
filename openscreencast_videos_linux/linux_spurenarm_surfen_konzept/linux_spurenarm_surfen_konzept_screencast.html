Spurenarm Surfen - Konzept
1. datensammelnde Webangebote meiden

2. Cookies, Javascript nur bei vertrauenswürdigen Websites

3. Werbung, HTML-Wanzen, Like-Buttons durch Filter blockieren

4. Browserinfos entfernen

5. risikoreiche Plugins, History, Geolocation deaktivieren

6. HTTPS-Zertifikate zusätzlich validieren

7. Anonymisierungsdienste nutzen

8. datensparsame Suchmaschinen nutzen
